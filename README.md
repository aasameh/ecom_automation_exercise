# E-commerce Test Automation

My test automation project for an e-commerce website using Selenium + pytest-bdd.

## What This Is

Automated tests for various e-commerce workflows using the Page Object Model pattern. Tests are written in Gherkin (BDD style) with step definitions in Python.

## Test Coverage

- User signup/login/logout flows
- Product browsing and detail pages
- Search functionality
- Contact form submission
- Cart subscriptions
- Test cases page verification

## Tech Used

- Selenium WebDriver
- pytest-bdd
- Python 3.12+
- Page Object Model design pattern

## Notes

- Feature files are in `tests/features/`
- Step definitions in `tests/steps/`
- Page objects in `pages/`
- Using Chrome with uBlock extension (`.crx` file)
